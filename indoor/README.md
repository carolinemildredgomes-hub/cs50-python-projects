# Indoor Voice

This project is part of the **CS50's Introduction to Programming with Python** course.

## Description
The program prompts the user for input and outputs the same text in **lowercase**.

It converts uppercase letters to lowercase while keeping punctuation, numbers, and whitespace unchanged.

## Example

Input:
HELLO

Output:
hello


Input:
THIS IS CS50

Output:
this is cs50


Input:
50

Output:
50

## Code

```python
text = input()
print(text.lower())
